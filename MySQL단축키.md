# MySQL Workbench 단축키

Ctrl + Shift + Enter: SQL 쿼리 실행
Ctrl + Enter: 쿼리 편집기에서 현재 쿼리 실행
Ctrl + S: 현재 열린 파일 저장
Ctrl + Shift + T: 새 쿼리 탭 열기
Ctrl + W: 현재 탭 닫기
Ctrl + Z: 실행 취소
Ctrl + Shift + Z: 다시 실행
Ctrl + F: 찾기
Ctrl + H: 바꾸기
F6: 다음 쿼리 탭으로 이동
Shift + F6: 이전 쿼리 탭으로 이동




# 커맨드 라인 인터페이스(CLI) 단축키

Ctrl + C: 현재 명령 취소
Ctrl + D: MySQL 세션 종료
Ctrl + L: 화면 클리어
Ctrl + R: 이전에 실행한 명령 검색
Arrow keys: 이전 명령들 사이를 이동
